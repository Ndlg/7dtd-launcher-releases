# NDLG 7DTD Launcher Releases

Public, signed binary releases for the NDLG 7DTD world and save launcher.

- `stable-*`: production releases
- `preview-*`: development tower-map and engine releases
- Windows and Linux bundles are offline and contain required Docker images.
- Every manifest is Ed25519/Minisign signed and every package is SHA-256 verified.

Source code and signing keys are intentionally not published in this repository.
